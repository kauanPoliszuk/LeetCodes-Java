## Sobre o Repositório

Este repositório contém minhas **soluções em Java** dos problemas resolvidos no **LeetCode** e **BeeCrowd**.

O principal objetivo é registrar meu progresso em **algoritmos**, **estruturas de dados** e **Competitive Programming**, além de servir como material de revisão e portfólio técnico.

---

## 🎯 Objetivos

- Fortalecer o conhecimento em Java e boas práticas de programação
- Dominar algoritmos e estruturas de dados clássicos
- Preparação consistente para entrevistas técnicas
- Manter um histórico organizado da minha evolução

---

## 📊 Status Atual

- **Linguagem principal:** Java (JDK 17+)
- **Plataformas:** BeeCrowd (foco atual) + LeetCode
- **Estrutura:** Organizado por plataforma
- **IDE:** IntelliJ IDEA
- **Foco:** Clean Code, Otimização de algoritmos e Complexidade

## 📌 Como as soluções estão organizadas

Cada pasta de problema geralmente contém:

- Código limpo e bem comentado
- Análise de complexidade | Big O(n) & Big O(1)
- Abordagem utilizada (quando relevante)

**Exemplo atual:** `src/BeeCrowd/1011/` → Resolução do problema BeeCrowd 1011.
